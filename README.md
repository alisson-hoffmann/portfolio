Portfólio pessoal com foco em UI/UX moderno: dark mode, glassmorphism e microanimações.

## Demo

> Visualização local: abra `index.html` no navegador ou sirva com um servidor estático.

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis e animações (Glassmorphism)
- JavaScript (ScrollReveal.js)

## Instalação / Como visualizar

Rápido usando `Live Server` extension no VS Code ou um servidor simples:

```bash
# usando Python 3
python -m http.server 3000

# abrir http://localhost:3000
```

Ou apenas abra `index.html` diretamente no navegador.

## Estrutura do projeto

```
primeiro trabalho/
├─ index.html
├─ about.html
├─ style.css
├─ main.js
└─ assents/ (imagens)
```

## Design & Features

- Dark mode sofisticado com paleta Slate + Cyan/Violet.
- Glassmorphism nas seções principais.
- Keyframes CSS: `floaty`, `glow` e transições suaves.
- Scroll interactions com ScrollReveal para revelar conteúdo em scroll.
